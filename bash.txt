Символы групповых операций:
*
?
[123] - один символ из перечисленных
[1-3] - один символ из диапазона

Выводит переменные окружения, например:
printenv PATH

Запускает команду с определенными переменными окружения, например эта выдаст ошибку:
env PATH=/ ls

Просмотр дерева процессов:
ps x -H

Просмотр man на русском языке, если таковая man-страница установлена в системе:
man -L ru adduser

Выводит все файлы в каталоге рекурсивно: 
find  /usr/local/ -exec ls -dlah -F --color {} \;

Выводит файлы с сортировкой по расширению:
ls -X

Аналоги where со своими особеностями:
command -v curl
type -p curl

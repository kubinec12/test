Adb Shell:
Показывает и устанавливает какие-то настройки:
settings

Gets an Android system property, or lists them all:
getprop

adb install - установка приложения

adb pm install - пакетный менеджер?

My APK - приложение для андроид для получения информации и совершения действий с приложениями и APK файлами, в том числе показывает подпись
ML Manager - приложение для андроид, сохраняет установленные приложения в APK-файлы
Droid Info - приложение для андроид, информация об устройстве
APKMirror Installer  - приложение для андроид, альтернативный установщик APK, позволяет увидеть причину при ошибке во время установки, могут быть сложности на MIUI
